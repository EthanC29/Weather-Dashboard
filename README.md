# Weather Forecast Dashboard

## Description
A Weather Forecast Dashboard that displays the weather conditions for the current day as well as the forecast for the next 5 days and varies based on the city name input in the search bar. It has a function to save any previous searches and allow the user to click them to go back to those results again.
